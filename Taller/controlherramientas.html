Configurar Supabase

Esta app usa Supabase como base de datos gratuita en la nube. Seguí estos pasos para conectarla:

1. Creá una cuenta en supabase.com
2. Creá un nuevo proyecto
3. Ejecutá el archivo control-herramientas-schema.sql en SQL Editor
4. Copiá la URL y la clave anónima desde Settings → API
5. Pegalas abajo y guardá

Copias de seguridad

Generá una copia de todos los datos de la base (herramientas, alumnos, profesores y retiros) en un archivo que podés guardar en tu computadora o pendrive. Si algún día se pierde la información, podés restaurarla desde ese archivo.

Descargar copia

El archivo JSON es la copia completa para restaurar. Los archivos CSV sirven para abrir las tablas en Excel.

Restaurar desde una copia

Seleccioná un archivo de copia (.json) generado por esta app. Se agregarán los registros que contenga. Atención: esto no borra los datos actuales, los complementa. Usalo solo si entendés lo que hacés.

Auditoría

Registro de actividad del sistema: ingresos, altas, ediciones y bajas. Elegí un rango de fechas y, si querés, filtrá por un usuario en particular.

⚠ El módulo de auditoría todavía no está activado. Ejecutá el archivo auditoria-schema.sql en Supabase (SQL Editor) y volvé a intentar.

Informes

Generá informes formateados y listos para imprimir o guardar como PDF. Elegí el tipo de informe, ajustá las opciones y tocá Generar. Después usá Imprimir / Guardar PDF para obtener el documento limpio.

Ajustes

Personalizá los nombres que aparecen en toda la aplicación. Útil si otra escuela, organización o sector quiere reutilizar el sistema. Los cambios se aplican para todos los usuarios.

⚠ El módulo de ajustes todavía no está activado. Ejecutá el archivo configuracion-schema.sql en Supabase (SQL Editor) y volvé a intentar.
