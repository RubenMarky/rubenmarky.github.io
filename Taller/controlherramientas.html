Configurar Supabase

Esta app usa Supabase como base de datos gratuita en la nube. Seguí estos pasos para conectarla:

1. Creá una cuenta en supabase.com
2. Creá un nuevo proyecto
3. Ejecutá el archivo control-herramientas-schema.sql en SQL Editor
4. Copiá la URL y la clave anónima desde Settings → API
5. Pegalas abajo y guardá
