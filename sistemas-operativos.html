1¿Qué es un Sistema Operativo?

Un sistema operativo (SO) es el software principal de una computadora. Se instala antes que cualquier otro programa y se encarga de hacer funcionar todo lo demás: el procesador, la memoria, el disco, el teclado, la pantalla y cualquier dispositivo conectado.

📌 Analogía Pensá en el SO como el director de una orquesta. No toca ningún instrumento, pero coordina a todos los músicos para que entren en el momento correcto. Del mismo modo, el SO no hace el trabajo de las aplicaciones, pero les da los recursos que necesitan.

El sistema operativo actúa como intermediario entre el usuario, las aplicaciones y el hardware. Los nombres de muchos SO incluyen el sufijo OS (Operating System): MS-DOS, iOS, macOS.

¿Qué hace el sistema operativo?

  • Administra el procesador (CPU): decide qué programa usa el CPU en cada momento.
  • Administra la memoria RAM: asigna y libera espacio para cada programa.
  • Gestiona el almacenamiento: organiza archivos y carpetas en el disco.
  • Controla la entrada y salida: comunica el teclado, el mouse, la impresora y otros dispositivos con las aplicaciones.
  • Mantiene la seguridad: protege el equipo mediante cuentas de usuario y permisos.

3Componentes del Sistema Operativo

El SO está formado por dos grandes grupos de programas:

Programas de control (el núcleo o kernel)

Coordinan el funcionamiento de todos los recursos del equipo. Se dividen en:

  • Gestión del procesador: controla el tiempo que cada proceso usa el CPU.
  • Gestión de memoria: asigna y controla el espacio en RAM y almacenamiento.
  • Gestión de entrada/salida: gestiona las operaciones con dispositivos externos (impresoras, lectores, etc.).
  • Gestión de datos: controla transferencia, apertura y acceso a archivos.
  • Gestión del sistema (supervisor): coordina y supervisa la seguridad general.

Programas de proceso

Ayudan a los programadores a escribir aplicaciones. Incluyen:

  • Traductores: compiladores, intérpretes y ensambladores. Convierten el código escrito por humanos a instrucciones que entiende la máquina.
  • Programas de servicio (utilidades): herramientas de mantenimiento y manipulación del sistema.

Evolución de los sistemas operativos

Nivel / ÉpocaCaracterísticas principales

Trabajo Práctico N° 1 · 1er Trimestre

Reconocimiento del equipo y administración del sistema

Hardware, sistema operativo, procesos y rendimiento usando las herramientas propias de Windows.

Objetivos

  • Reconocer las características de hardware y SO del equipo.
  • Identificar la edición y arquitectura de Windows instalado.
  • Comprender el concepto de proceso y la gestión que hace el SO.
  • Usar Configuración, Administrador de tareas y Explorador de archivos.

Parte 1 — Sistema operativo y CPU

📌 Cómo acceder: Clic derecho en Inicio → Sistema. O bien: Configuración (Win + I) → Sistema → Información.

Completar la siguiente información del equipo:

  • Edición de Windows instalada (Home, Pro, Education…)
  • Versión y compilación del sistema operativo
  • Tipo de sistema: 32 o 64 bits
  • Nombre del equipo
  • Modelo y marca del procesador (CPU), velocidad en GHz
  • Cantidad de núcleos físicos y procesadores lógicos
  • Memoria RAM instalada y utilizable (en GB)

Parte 2 — Almacenamiento

📌 Cómo acceder: Explorador de archivos (Win + E) → Este equipo → clic derecho en disco C: → Propiedades.
  • Tipo de unidad (HDD o SSD)
  • Capacidad total, espacio ocupado y disponible (en GB)
  • Sistema de archivos (NTFS, FAT32…)
  • Tamaño de carpetas: C:\Windows, Archivos de programa, Usuarios

Parte 3 — Procesos y rendimiento

📌 Cómo abrir: Ctrl + Shift + Esc → pestaña Rendimiento y pestaña Procesos.
  • % de uso actual de CPU, RAM y disco
  • Cantidad total de procesos en ejecución
  • Los 3 procesos que más CPU consumen (nombre y %)

Parte 4 — Reflexión

  • ¿Qué diferencia encontrás entre un disco HDD y un SSD? ¿Cuál tiene el equipo?
  • ¿Cuál de los procesos en ejecución consume más recursos? ¿Por qué creés que es así?

5Gestión de archivos y carpetas

Un archivo es un conjunto de información guardada con un nombre y una extensión. La extensión (las letras después del punto) indica el tipo de archivo y con qué programa se abre.

Una carpeta (o directorio) es un contenedor que organiza archivos y otras carpetas. La ruta indica la ubicación exacta: C:\Usuarios\Ana\Documentos\tarea.docx

7Redes y conectividad

Una red informática es un conjunto de dispositivos conectados entre sí para compartir información y recursos. El ejemplo más grande es Internet, una red mundial formada por millones de redes más pequeñas.

💡 Internet ≠ Web: Internet es la red física y de protocolos. La Web (World Wide Web) es solo uno de los servicios que viajan por internet — las páginas que vemos en un navegador. Correo, videollamadas y juegos online también usan internet pero no son la Web.

Tipos de redes

SiglaNombreEjemplo cotidiano
PANRed personalCelular conectado a auriculares Bluetooth.
LANRed de área localLas PCs del aula conectadas al mismo router.
MANRed metropolitanaVarias sedes de una universidad en la misma ciudad.
WANRed de área ampliaInternet. Conecta países y continentes.

Dispositivos de red

DispositivoFunción
MódemConecta el hogar con el proveedor de internet (ISP). Convierte la señal de la calle.
RouterDistribuye internet entre los dispositivos de la red, por cable o Wi-Fi.
SwitchConecta varios equipos por cable en una red local (aulas, oficinas).
Access pointCrea o amplía una red Wi-Fi en zonas grandes.
RepetidorToma la señal Wi-Fi del router y la extiende para llegar más lejos.

Dirección IP y DNS

Cada dispositivo en una red tiene una dirección IP que lo identifica, como un número de documento. Hay dos tipos:

  • IP privada: válida solo dentro de tu red local (ej.: 192.168.1.5). Solo la ven los dispositivos de tu casa u oficina.
  • IP pública: única en el mundo. La asigna tu proveedor de internet y es la que ven los sitios web.

El DNS (Domain Name System) traduce los nombres que escribís (google.com) a la dirección IP real del servidor. Sin DNS habría que memorizar números para entrar a cada sitio.

Objetivos

  • Leer la configuración de red del equipo con ipconfig.

Actividad 1 — Tipos de redes

  • Tu celular conectado a auriculares Bluetooth.
  • Las PCs del aula conectadas al mismo router.

Ejecutar ping google.com y ping 8.8.8.8. Anotar si hay respuesta y el tiempo promedio en ms. ¿Qué conclusión sacás si uno funciona y el otro no?

Entrar a speedtest.net o fast.com y anotar: velocidad de descarga (Mbps), velocidad de subida (Mbps) y latencia (ms).

8Comparativa de Sistemas Operativos

Los tres sistemas operativos de escritorio más usados son Windows, Linux y macOS. Cada uno tiene ventajas e inconvenientes según el contexto de uso.

Ventajas de cada SO

🪟 Windows
  • El más utilizado del mundo: gran cantidad de aplicaciones disponibles.
  • Amplia compatibilidad con hardware del mercado.
  • Interfaz gráfica intuitiva y fácil de usar.
🐧 Linux
  • Estabilidad: se cuelga mucho menos que Windows.
  • Gratuito, sin costo de licencia.
  • Gran cantidad de software libre disponible.
🍎 macOS
  • Seguridad y estabilidad reconocidas.
  • Interfaz simple y diseño elegante.
  • Ideal para diseño gráfico y edición multimedia.

Inconvenientes de cada SO

🪟 Windows
  • Mayor riesgo de virus y malware.
  • Problemas de estabilidad (cuelgues, pantalla azul).
  • Errores de compatibilidad entre versiones.
🐧 Linux
  • Exige mayores conocimientos informáticos.
  • Menos software comercial disponible.
  • Soporte más limitado para hardware nuevo.
🍎 macOS
  • Mayor costo: el hardware Apple es caro.
  • Menos opciones de software que Windows.
  • Mayor dificultad para obtener soporte técnico local.

10Glosario de términos clave

Sistema operativo (SO)
Software principal que administra el hardware y permite que las aplicaciones funcionen. Intermediario entre el usuario, los programas y el equipo físico.
Kernel (núcleo)
Parte central del SO que concentra las funciones más básicas: gestión del procesador, memoria, entrada/salida y seguridad.
Proceso
Programa en ejecución. El SO le asigna tiempo de CPU, memoria y otros recursos.
RAM (Random Access Memory)
Memoria de acceso aleatorio. Almacena los datos y programas en uso. Es volátil: se borra al apagar el equipo.
HDD / SSD
Tipos de unidades de almacenamiento permanente. HDD usa discos magnéticos; SSD usa memoria flash y es más rápida y silenciosa.
Archivo
Conjunto de información guardada con un nombre y una extensión que indica su tipo.
Extensión
Las letras después del último punto en el nombre de un archivo (ej.: .docx, .mp4). Indican el tipo y el programa que lo abre.
Ruta (path)
Dirección que indica la ubicación exacta de un archivo dentro del sistema (ej.: C:\Usuarios\Ana\tarea.docx).
CMD (Símbolo del sistema)
Terminal clásica de Windows heredada de MS-DOS. Se controla escribiendo comandos de texto.
PowerShell
Terminal moderna de Windows. Usa cmdlets con la estructura Verbo-Sustantivo (ej.: Get-Process). Más potente que CMD para automatización.
Dirección IP
Número único que identifica a un dispositivo en una red. Las IP privadas son locales; las públicas se ven desde internet.
DNS (Domain Name System)
Sistema que traduce nombres de dominio (google.com) a direcciones IP. Es el "directorio telefónico" de internet.
Router
Dispositivo que distribuye internet entre los equipos de una red local, por cable (Ethernet) o de forma inalámbrica (Wi-Fi).
LAN / WAN
Red de área local (un edificio o vivienda) y red de área amplia (varios países o el mundo entero).
Protocolo
Conjunto de reglas que los dispositivos siguen para comunicarse. Ej.: HTTP para páginas web, SMTP para correo.
